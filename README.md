\# Digit Recognizer con MNIST



Questo progetto è un'applicazione in Python per riconoscere cifre scritte a mano utilizzando un modello MLP (Multi-Layer Perceptron) addestrato sul dataset MNIST. L'interfaccia grafica è realizzata con \*\*Tkinter\*\* e permette di disegnare un numero a mano libera per ottenere la predizione del modello.



---



\## Contenuti della repository



\- `main\_numpy.py` – il file principale dell'applicazione.

\- `dataset/` – cartella contenente eventuali modelli pre-addestrati e il file requirements.txt.

\- `requirements.txt` – elenco delle librerie Python necessarie.

\- `.gitignore` – per escludere file e cartelle non necessari (come l'ambiente virtuale `venv311`).

\- `README.md` – questa documentazione.
